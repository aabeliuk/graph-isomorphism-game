# Graph Isomorphism Puzzle

A visual puzzle game to explore graph isomorphism. The player must drag nodes to match two isomorphic graphs. Built using `pygame` and `networkx`.

## Requirements

- Python 3.x
- pygame
- networkx

## Installation

```bash
pip install pygame networkx
